# AdoptMe - Backend API 🐾

Proyecto final para el curso de Backend III. Esta API RESTful gestiona la adopción de mascotas, implementando arquitectura por capas, testing funcional y despliegue contenerizado.

## 🚀 Tecnologías Utilizadas
* Node.js & Express
* MongoDB & Mongoose
* Jest & Supertest (Testing)
* Docker & DockerHub
